# rest-fight

### Module de gestion du combat entre deux pokemons.

---

- Build de l'application
  ```shell
    mvn clean package
  ```

- Start de l'application sans les tests
  ```shell
    mvn clean package -DskipTests quarkus:dev
  ```

---

- Accès à la DevUI
    - http://localhost:8030/q/dev-ui/
